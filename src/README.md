# 101401017_comp3123_labtest2

Simple React Weather App for COMP3123 Lab Test 2.

## Tech

- React (Create React App)
- OpenWeatherMap Current Weather API

## Features

- Search weather by city name
- Shows temperature, description, humidity, wind, min/max, feels-like
- Uses weather icon and themed UI

## Run locally

```bash
npm install
npm start
